# Claude Code Actions Sandbox

## 実行方法

### 前提条件
- Node.js (バージョン 18 以上)
- Git

### 実行手順
1. リポジトリをクローン:
   ```bash
   git clone https://github.com/inotsuyo/ClaudeCodeAction-test.git
   cd ClaudeCodeAction-test
   ```

2. 依存関係をインストール:
   ```bash
   npm install
   ```

3. アプリケーションを実行:
   ```bash
   npm start
   ```
